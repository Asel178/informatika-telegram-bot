# informatika-telegram-bot
# 🤖 Сабаққа арналған интерактивті Telegram-бот

Бұл — мектептегі информатика сабақтарында оқушылардың білімін тексеруге және қызықты тапсырмалар орындатуға арналған цифрлық жоба.

## 🎯 Жобаның мақсаты
Оқушыларға Python бағдарламалау тілін ойын түрінде, интерактивті батырмалар арқылы үйрету.

## ⚙️ Боттың негізгі функциялары
* `/start` — Ботты іске қосу және сәлемдесу.
* `Тапсырмалар` батырмасы — Оқушыларға арналған деңгейлі сұрақтар.
* `Пайдалы сілтемелер` — Сабаққа қатысты қосымша материалдар мен Stepik курсына сілтеме.

## 🛠 Қолданылған технологиялар
* Python 3.x
* Telebot (немесе aiogram) кітапханасы
